从 MySQL Shell 8.0.17 开始，您可以通过在启动时加载的用户定义插件来扩展 MySQL Shell。插件可以用 JavaScript 或 Python 编写，它们包含的函数在 MySQL Shell 的 JavaScript 和 Python 模式下都可用。
